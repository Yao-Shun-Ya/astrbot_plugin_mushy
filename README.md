# AstrBot 插件：特定人触发图文回复

> 模块名称: astrbot_plugin_mushy
> 寄语：爻舜和鲸鱼小憩要长长久久。

## 📝 模块简介

本插件是一款为**特定用户 ID** 设计的高度定制化回复模块。它通过读取 WebUI 配置的关键词，对白名单内的用户消息进行精准匹配，并回复预设的图片与文本，实现专属、甜蜜的自动化互动。

### ✨ 核心功能与特性

本模块支持完全可视化配置，具备以下主要特性：

* **用户白名单机制**
    * 插件仅对配置中列出的 QQ/用户 ID 生效，非白名单用户触发指令将被忽略。
* **多规则分组管理**
    * 支持在 WebUI 中通过 **`触发规则配置组`** 管理两条条独立的触发规则（如 `rule_1`、`rule_2` 等）。
* **复合消息响应**
    * 触发后立即发送由自定义图片和回复文本构成的**复合消息链**，实现图文并茂的专属回应。
* **WebUI 可视化配置**
    * 关键词、图片路径和回复文本等所有参数均可通过 AstrBot WebUI 进行调整，无需修改代码。
